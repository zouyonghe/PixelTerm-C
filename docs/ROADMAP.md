# PixelTerm-C Roadmap

## Current Status
Core functionality is complete and stable as of v1.4.0. Ongoing work focuses on
terminal compatibility and rendering quality across SSH sessions.

## Near-Term Ideas
- Improve terminal capability detection and fallbacks
- Expand automated testing coverage
- Improve performance profiling and diagnostic output

## Longer-Term Ideas
- Configuration file support
- Advanced image processing filters
- Additional terminal-specific optimizations
