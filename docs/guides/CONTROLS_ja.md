# 🎮 操作

*[English](CONTROLS.md) | [中文](CONTROLS_zh.md) | 日本語*

### 基本操作

| キー | 機能 |
|-----|------|
| ESC | アプリを終了 |
| Ctrl+C | 強制終了 |

### マウス操作

モードごとの移動や選択では、マウス操作も利用できます。

| 操作 | 機能 | 対象モード | 補足 |
|------|------|------------|------|
| 左クリック | 次の画像へ進む | 画像ビュー（単体表示） | 動画では再生 / 一時停止になります。 |
| 左ダブルクリック | グリッドプレビューへ切り替え | 画像ビュー（単体表示） | |
| マウスホイール上下 | 画像を拡大 / 縮小（カーソルが画像上にある場合） | 画像ビュー（単体表示） | 動画 / アニメーション画像では使えず、画像領域外では無視されます。 |
| 左クリック | 画像を選択 | グリッドプレビュー | カーソル下の画像を選択します。 |
| 左ダブルクリック | 選択中の画像を画像ビューで開く | グリッドプレビュー | カーソル位置の画像を開きます。 |
| マウスホイール上下 | 前 / 次のページへ移動 | グリッドプレビュー | ページ単位で画像を切り替えます。 |
| 左クリック | ページを選択 | 電子書籍プレビュー | カーソル下のページを選択します。 |
| 左ダブルクリック | 選択中のページを電子書籍ビューアで開く | 電子書籍プレビュー | カーソル位置のページを開きます。 |
| マウスホイール上下 | 前 / 次のページへ移動 | 電子書籍プレビュー | ページ一覧を切り替えます。 |
| 左クリック | 次のページへ進む | 電子書籍ビューア | レイアウトに応じて 1 ページまたは 2 ページ進みます。 |
| 左ダブルクリック | 電子書籍プレビューへ切り替え | 電子書籍ビューア | |
| マウスホイール上下 | 前 / 次のページへ移動 | 電子書籍ビューア | |
| 左クリック | 項目を選択 | ファイルマネージャ | カーソル下のファイルまたはディレクトリを選択します。 |
| 左ダブルクリック | 選択中の項目を開く（ディレクトリ / ファイル） | ファイルマネージャ | ディレクトリに入るか、画像を開きます。 |
| マウスホイール上下 | 項目を上下に移動 | ファイルマネージャ | ファイル / ディレクトリ一覧をスクロールします。 |

### 画像ビュー（単体表示）

単体の画像を開いたときの基本モードです。

| キー | 機能 |
|-----|------|
| ←/→ | 前の画像 / 次の画像 |
| ↑/↓ | 前の画像 / 次の画像 |
| h/k | Vim スタイル移動（前の画像） |
| l/j | Vim スタイル移動（次の画像） |
| Enter | グリッドプレビューへ切り替え |
| TAB | 画像ビュー / グリッドプレビュー / ファイルマネージャを切り替え |
| i | 画像情報の表示を切り替え |
| `~` / `` ` `` | Zen モード（UI テキストの表示 / 非表示）を切り替え |
| r | 現在の画像を削除 |
| d/D | ディザリングの ON / OFF を切り替え |

### 動画再生

画像ビューで現在の項目が動画のときに使える操作です。

| キー | 機能 |
|-----|------|
| 左クリック | 再生 / 一時停止 |
| Space | 再生 / 一時停止 |
| ←/→ | 5 秒巻き戻し / 5 秒早送り |
| ↑/↓ | 前 / 次のメディアへ切り替え |
| F | FPS 表示を切り替え |
| + / - | 動画の表示倍率を調整（`+` で拡大、`-` で縮小） |
| p / P | 動画プロトコルを切り替え: `text -> sixel -> iterm2 -> kitty -> text` |

### グリッドプレビュー（サムネイル表示）

複数の画像サムネイルをグリッドで表示するモードです。

| キー | 機能 |
|-----|------|
| ←/→ | 選択を左 / 右へ移動 |
| ↑/↓ | 選択を上 / 下へ移動 |
| h/j/k/l | Vim スタイル移動（左 / 下 / 上 / 右） |
| PgUp/PgDn | ページ切り替え |
| Enter | 選択中の画像を画像ビューで開く |
| TAB | 画像ビュー / グリッドプレビュー / ファイルマネージャを切り替え |
| `~` / `` ` `` | Zen モード（UI テキストの表示 / 非表示）を切り替え |
| r | 選択中の画像を削除 |
| d/D | ディザリングの ON / OFF を切り替え |
| +/= | 拡大 |
| - | 縮小 |

### 電子書籍ビューア

| キー | 機能 |
|-----|------|
| ←/→ | 前のページ / 次のページ |
| ↑/↓ | ページ送り / ページ戻り（単ページ: 1、見開き: 2） |
| h/j/k/l | Vim スタイル移動（左 / 下 / 上 / 右） |
| PgUp/PgDn | ページ送り / ページ戻り |
| P | ページジャンプ（数字入力後、Enter で確定、P/ESC でキャンセル） |
| Enter | 電子書籍プレビューへ切り替え |
| TAB | ファイルマネージャへ戻る |
| `~` / `` ` `` | Zen モード（UI テキストの表示 / 非表示）を切り替え |

注: 電子書籍ビューは、端末のアスペクト比に応じて単ページ表示と見開き表示を自動で切り替えます。

### 電子書籍プレビュー（ページグリッド）

| キー | 機能 |
|-----|------|
| ←/→ | 選択を左 / 右へ移動 |
| ↑/↓ | 選択を上 / 下へ移動 |
| h/j/k/l | Vim スタイル移動（左 / 下 / 上 / 右） |
| PgUp/PgDn | ページ切り替え |
| P | ページジャンプ（数字入力後、Enter で確定、P/ESC でキャンセル） |
| Enter | 選択中のページを電子書籍ビューアで開く |
| TAB | 選択中のページを電子書籍ビューアで開く |
| +/= | 拡大 |
| - | 縮小 |

### ファイルマネージャ

このモードではディレクトリとファイルをブラウズできます。ここでは `h/j/k/l` は Vim スタイル移動ではなく、先頭文字ジャンプ用のキーとして扱われます。

| キー | 機能 |
|-----|------|
| ←/→ | 親ディレクトリへ戻る / 選択中のディレクトリまたはファイルを開く |
| ↑/↓ | 項目を上下に移動 |
| Enter | 選択中のディレクトリまたはファイルを開く |
| TAB | 画像プレビュー / 電子書籍プレビューを開く、または戻る |
| Backspace | 隠しファイルの表示 / 非表示を切り替え |
| 任意の英字キー (a-z/A-Z) | その文字で始まる次の項目へジャンプ |
